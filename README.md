# Rndstorm
## By Barrie Millar
### A Progressive Web App (PWA) for generating random numbers on mobile and desktop devices 

## Introduction

Rndstorm is a Progressive Web App (PWA) for generating random numbers on mobile and desktop devices. It is designed to be installable and thus run offline on mobile devices.
